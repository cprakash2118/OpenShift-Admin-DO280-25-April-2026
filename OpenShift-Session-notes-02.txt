Prod env:- OCP cluster 
 - 3 master nodes  (Master node alway follow old no)
 - N worker nodes (Based on the workload)
 FYI- 3 master can handle almost 2k worker nodes (w.r.t OCPv4.6 - current version 4.21).
 
 Master nodes -> K8s/OCP services 
 Worker nodes -> End users application (Banking etc)
 
FYI- Master node also can be worker nodes (But not good for prod).

OCP cluster:-
master node -> control plane 
worker node -> compute node
